# URL Shortener

Un acortador de URLs potente y moderno, diseñado para ofrecer estadísticas detalladas y una gestión sencilla de enlaces.

## Características

- **Acortamiento de URLs**: Crea enlaces cortos y fáciles de compartir.
- **Estadísticas Detalladas**: Visualiza visitantes, países, navegadores, sistemas operativos y referrers en tiempo real.
- **Gráficos Profesionales**: Visualización de datos mediante gráficos interactivos de líneas y barras.
- **Panel de Administración**: Gestión completa de tus enlaces acortados.
- **Diseño Premium**: Interfaz moderna, rápida y totalmente responsiva.

## Instalación y Configuración Local

1. **Clonar el repositorio**:
   ```bash
   git clone [url-del-repositorio]
   cd url-shotener
   ```

2. **Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Configurar variables de entorno**:
   Crea un archivo `.env.local` basado en el ejemplo proporcionado y añade tus credenciales.

4. **Ejecutar en desarrollo**:
   ```bash
   npm run dev
   ```

   El proyecto estará disponible en [localhost:3000](http://localhost:3000/).

## Tecnologías Utilizadas

- **Framework**: React / Framework Moderno.
- **Gráficos**: Recharts.
- **Estilos**: Tailwind CSS.
- **Iconos**: Lucide React.
- **Componentes**: shadcn/ui.
